Скрипт установки и настройки IGMP Proxy на роутеры с OpenWrt. Проверено с iptv-приставками на МТС.

# Установка
Заходим на роутер через ssh, скачиваем и запускаем установочный скрипт:
```
wget -O install-igmp.sh https://raw.githubusercontent.com/wwwean/igmpproxy-install/refs/heads/main/install.sh && chmod +x install-igmp.sh && ./install-igmp.sh
```
